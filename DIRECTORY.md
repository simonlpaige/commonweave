# The Movement Visible to Itself

> *"I knew that if we could understand the connections and visualize the breadth of global efforts on behalf of social and environmental justice, we would recognize the largest movement the world has ever seen."*
> -- Paul Hawken, WiserEarth founder, 2007

This directory maps organizations that are already doing the work of Ecolibrium's framework -- building the post-scarcity commons, governing shared resources, redistributing power, restoring ecosystems -- without using that language or knowing they are part of the same movement.

A rural electric cooperative in Kentucky doesn't call itself "community-owned energy commons." A restorative justice circle in Cape Town doesn't call itself "post-carceral conflict resolution infrastructure." A seed library in Bangalore doesn't call itself "biodiversity commons governance." But they are all expressions of the same underlying project.

This directory is the mirror WiserEarth tried to build. It is incomplete by design. Add to it.

**Format:** Each entry includes what the org does, its scale, its URL (where verified), and 2-3 taxonomy tags. Entries marked *[working unknowingly]* are organizations that fit the framework's goals but would not recognize themselves in its language -- the gap between what they do and what we call it is worth naming.

**Status:** Living document. Entries marked [VERIFY] need a human to confirm current operational status.

## How to Use This Directory

**For outreach:** These organizations are the network. Many will not know Ecolibrium exists. Many would recognize their work in its framework if shown. The outreach templates in OUTREACH.md apply here.

**For research:** Each section has existing organizations demonstrating the framework's proposals in practice. Before proposing anything, check whether it's already being done -- and at what scale, with what results.

**For the "working unknowingly" entries specifically:** These are the highest-value connection opportunities. An organization that is already doing the work but doesn't have the language for it is not a convert -- it's an ally who hasn't been introduced to the rest of the movement yet.

**To add an entry:** Open a Pull Request with the org name, description, URL, tags, and whether it's "working unknowingly." See CONTRIBUTING.md for format.
